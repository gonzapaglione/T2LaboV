Tim Berners-Lee es un científico británico conocido por haber inventado la World Wide Web en 1989.
Mientras trabajaba en el CERN, desarrolló los conceptos de URL, HTTP y HTML,
creando así el primer navegador web y el primer sitio web.
Su trabajo revolucionó la forma en que accedemos y compartimos información en Internet.

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web.
Permite crear páginas con texto, imágenes, enlaces, videos, formularios, entre otros elementos.
Es un pilar fundamental del desarrollo web, junto con CSS y JavaScript.
